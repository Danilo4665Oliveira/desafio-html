Início de tudo

Os computadores nasceram em ambiente de guerra com intuito de trazer vantagem para o seu respectivo país.
Alguns desses computadores da chamada primeira geração foram:

  • Harvard Mark I (1944)
  • Colossus(1946)
  • Eniac(1946)

A partir daí com o passar das gerações os computadores foram de grandes máquinas que ocupavam uma sala inteira a apenas um computador pessoal (Personal computer), dando origem ao famoso termo: PC.

Internet - Como surgiu?

voltar

A internet surgiu, na década de 1960, como uma rede (chamada Arpanet) para envio de informações entre centro de pesquisas e instalações militares e o Pentágono. A primeira mensagem enviada se deu entre computadores da UCLA e da SRI, instituições universitárias dos Estados Unidos.

Tag: "a"

voltar

A tag a , conhecida como tag de âncora, é responsável por vincular os demais documentos que o nosso HTML possui, links internos e externos. Para conseguirmos declarar essa tag, devemos incluir o atributo href e uma frase ou palavra dentro da tag a.

Tag:"p"

voltar

O elemento Parágrafo. O elemento HTML

representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco.

Tag's: "h1/2/3/4/5/6"

voltar

Heading tags, também conhecidas como tags de cabeçalho, são elementos HTML usados para estruturar e organizar o conteúdo em páginas da web. Elas são representadas pelos elementos h1, h2, h3, h4, h5 e h6, onde h1 é o cabeçalho mais importante e h6 é o menos importante.

Tag's: "ul" e "ol":

voltar

Resumo. O elemento HTML li (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista. Este item deve estar contido em um elemento pai: uma lista ordenada ( ol ), uma lista desordenada ( ul ) , ou um menu ( menu ) e representa uma única entidade dessa lista.
Exemplos:

  • Lista
  • Não
  • Ordenada
  1. Lista
  2. Ordenada

Links

voltar

O href é um atributo da tag de âncora (de links), ou seja, o "a" . Essas tags precisam desse atributo para especificar uma localização, ou seja, para qual link você deve ir. Significa que queremos ir para uma seção da página que contém o ID 'contato'.
